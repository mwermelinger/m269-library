lib.digraph module

An implementation of weighted directed graphs.

class lib.digraph.DirectedGraph

Bases: object

Implement a directed graph (digraph).

Each node must be a hashable value: a number, string, tuple of hashable values, etc. Each edge has a direction, i.e. it goes from a node to another. The two nodes must be different. Each edge has a weight, which must be a positive number. If all weights are the same, the graph is effectively unweighted.

add_edge(source, target, weight=1)

Add a directed edge from source to target with the given weight.

Replace the weight value if the edge already exists. Add any node that doesn’t exist. Assume the two nodes are different. Assume the weight is a positive number. Assume the nodes are of a hashable type.

add_node(node)

Add the node to the graph.

Do nothing if the graph already has the node. Assume the node is of a hashable type.

has_edge(source, target)

Check if the graph has a directed edge from source to target.

Return False if the edge or either node don’t exist, otherwise True.

has_node(node)

Return True if the graph contains the node, otherwise False.

neighbours(node)

Return the set of neighbours of node.

Assume the graph has the node.

nodes()

Return the set of all nodes in the graph.

remove_edge(source, target)

Remove the edge from source to target from the graph.

Do nothing if the edge doesn’t exist.

remove_node(node)

Remove the node and its edges from the graph.

Do nothing if the node doesn’t exist.

shortest_path(start, end)

Compute a shortest path from the start to the end node.

A shortest path is a path of minimal total weight. Return a list of nodes forming a path from start to end. Return the empty list if there is no path. Assume the graph has the start node.

topological_sort()

Return a list of nodes, in topological order.

Return the empty list if the graph is cyclic.

unweighted_distance(start, end)

Return the smallest number of edges to go from start to end.

Return infinity if there is no path from start to end. Assume the graph has the start node.

unweighted_edges()

Return the set of all edges in the graph, without weights.

An edge is a tuple (source, target).

visited_bfs(start)

Do a breadth-first search of the graph, from the start node.

Return a list of nodes in visited order, the first being start. Assume the start node exists.

visited_dfs(start)

Do a depth-first search of the graph, from the start node.

Return a list of nodes in visited order, the first being start. Assume the start node exists.

weight(path)

Return the total weight of the path.

A path is a list of nodes of the graph [n1, n2, …] such that n1-n2, n2-n3, etc. are edges in the graph. Return 0 if the path has a single node. Return infinity if the path is empty or invalid.

weighted_distance(start, end)

Return the minimal total weight to go from start to end.

Return infinity if there is no path. Assume the graph has start node.

weighted_edges()

Return the set of all edges in the graph, with weights.

An edge is a tuple (source, target, weight).
